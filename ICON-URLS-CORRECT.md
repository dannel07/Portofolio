## 🎯 Cara Cepat Cari Icon:

1. Buka: https://simpleicons.org/
2. Search nama teknologi
3. Copy slug (nama teknologi dalam URL)
4. Format: `https://cdn.simpleicons.org/[slug]/[hex-color]`

Contoh:
- Next.js → slug: `nextdotjs`
- Node.js → slug: `nodedotjs`
- Vue.js → slug: `vuedotjs`

---

## 📝 Tips:

- Gunakan warna resmi teknologi (lihat di simpleicons.org)
- Nama dengan titik (.) diganti jadi `dot` → next.js = nextdotjs
- Nama dengan spasi diganti jadi lowercase → Visual Studio = visualstudio
- Kalau icon tidak muncul, cek nama slug di simpleicons.org
